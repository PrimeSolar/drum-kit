## Features

- Play drums clicking on the images or using keyboard keys: W, A, S, D, J, K, L.
- Responsive design for various screen sizes.
- Easy to extend with additional sounds and features.
- Clean user interface.

## Usage

Press the keys W, A, S, D, J, K, L on your keyboard or click on the images to play different drum sounds. Happy drumming! ü•Å
